import { Component,Input,input } from '@angular/core';

-Input : with big I = it is decorator 
-input : with small I =  it called :special function from single. 

Ex: 

- avatar = input(); it tell the Angular this avatar is the property that get the input from the component 
- name=input(''); it assign the initial value to the name 
- avatar =input<string>(); tell the TS that our input is String (typescripe that tell this is the generice type)

syntax : input<T>() 

==> avatar =input<string>(); avatar property will whole the InputSignal in the end and have string input 

Note:

- avatar =input.required<string>(); we use required : mean it need have input and the type is string 
- but avatar =input.required<string>(''); no working because no need to ' ' for the initial value .
